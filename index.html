Welcome!

The CFPB is a federal agency dedicated to ensuring that markets for consumer financial products and services are fair, transparent, and competitive. To support this mission, technologists at the CFPB build external resources for the public and internal tools to help the Bureau run more efficiently. We'd like to share our work with you.

Our code

qu

An in-progress data platform created to serve our public data sets.

hmda-tools

Tools to make importing and analyzing mortgage data easier.

regulations-parser

All the code to parse a regulation.

idea-box

An application that lets an organization collect ideas, comment on them, and vote them up.

ec2mapper

EC2mapper is a web application that provides a user-friendly interface to view Amazon AWS network configurations, while allowing changes to be easily tracked over time.

transit_subsidy

Open source version of a simple Transit Subsidy intake form that is commonly used by United States federal government agencies. Home of the first citizen pull request accepted by the U.S. government!

django-nudge

Gently push content from development to production.

django-cache-tools

Tools to make caching easier in Django.
